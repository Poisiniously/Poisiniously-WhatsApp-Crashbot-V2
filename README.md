## Quickstart
You can't skip the [wiki](https://github.com/Poisiniously/Poisiniously-WhatsApp-Bot-V2/wiki/Getting-started)!

Or just do this:
1. Make sure you have [Node.js](https://nodejs.org/en/download) (Version 16 or higher) installed on your system.

2. Open your terminal inside the project folder and run the following command to install the necessary npm packages:
```
npm install @whiskeysockets/baileys pino qrcode-terminal
```
or just click install.bat on Windows or install.sh on MacOS.

3. Run the main file using Node.js:
```
node index.js
```
or click start.bat on windows.

4. Just connect your WhatsApp and have fun!
